fileSYSman is a small program that manages common document files for you.

On first execution, the program will check for the config file needed.
Don't panic, it will tell you to create one for you and will ask for required paths for the program to run.
When you're done with the configuration, you can edit later your prefered path in the txt file.
If you entered a misstyped path, it will check whether it exists and stops before it moves your files.
Once, its passed through the conditions, you can tap any key to continue and do the magic for you.

REQUIREMENTS:
Operating System: The program is designed to run on Windows, Linux, or macOS, as long as the system supports C++17 or later.
C++ Compiler: A C++17-compatible compiler such as GCC, Clang, or MSVC is required to compile and run the code.
